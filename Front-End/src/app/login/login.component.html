This Website is built in Angular

And it is fueled by dynamic API data from https://swapi.py4e.com/

Register, login and explore starships, characters, and Star Wars films in an immersive, interactive experience.

It uses JSON-Server

to simulate a RESTful API using a JSON file as the data source.

As a Front-End Developer, using JSON-Server comes very handy to create mock APIs

Some of its Key Features:

  • User Login and Registration
  • Routes protection
  • Use of Observables for API consumption
  • Infinite scroll
